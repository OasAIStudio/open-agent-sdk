Open Agent SDK is a TypeScript SDK for building production-grade AI agents with tool use, session persistence, permissions, hooks, and multi-provider support.

This docs site is organized around:

- API reference for prompt/session/provider usage
- TypeScript type reference for strict integration
- Migration guidance from Claude Agent SDK
